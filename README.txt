Commands to run on siue servers
g++ ShortestSuperstring.cpp
./a.out

Upon program execution...
The user will be prompted to enter the name of the file
they would like to run. When a valid filename is entered,
the program will then execute the algorithm to find the 
shortest supersting, printing out the alignment matrix, 
followed by the space augmented alignments of the dna 
fragments, and ending with the superstring of those 
fragments being printed.
